# PIM 3 - Plataforma de Cursos

# Sobre o projeto
Sistema de plataforma de cursos online com foco em desempenho e gamificação.

# Objetivo
Desenvolver um sistema que permita:
- Gerenciamento de usuários
- Acompanhamento de desempenho
- Sistema de gamificação

# Tecnologias
- C#
- .NET
- Git e GitHub
- Astah
- Figma

# Estrutura do projeto
- docs → documentação
- diagramas → UML
- codigo → implementação
- design → protótipos e interfaces
